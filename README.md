# pac2fasta
bwa index to fasta

## example
```
.
├── pac2fasta.py
├── test.fa.amb
├── test.fa.ann
├── test.fa.bwt
├── test.fa.pac
└── test.fa.sa
```

* run:
```bash
python3 pac2fasta.py test.fa
```

* result:
```
.
├── pac2fasta.py
├── test.fa
├── test.fa.amb
├── test.fa.ann
├── test.fa.bwt
├── test.fa.pac
└── test.fa.sa
```
